Ansible Role: Win-Drivers-Rutoken-eToken
=========

Эта роль установит Драйвера eToken x64 и Рутокен x64 на ОС Windows.
Драйвера eToken Ver=5.1
Драйвера Rutoken Ver=4.8.10.0
